## Quickstart

1) Backend starten

```powershell
cd backend
npm install
npm run dev
```

2) Frontend starten

```powershell
cd frontend
npm install
npm run dev
```

Backend: `http://localhost:4000`
Frontend: `http://localhost:5173`
